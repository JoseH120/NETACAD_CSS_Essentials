Customizing forms with CSS

Styling text inputs and labels

  • The .custom-form class style the form container with a maximum width, centered alignment, padding, and background-color.
  • The label elements are displayed as block elements, which means they appear on a new line, and they are styled with color and margin for spacing.
  • Text inputs (<input type='text'> and <input type='email'>) are styled to take full width of the form container. Padding, border, and border-radius are applied to polished, user-friendly appearance.

Code explanation

  • A div element the class .button-spacer is introduced between the last input field and the Submit button. This div serves as a spacer, creating visual separation and enhacing the form's layout.
  • The .button-spacer class is styled with a height property, determining the amount of space between the last input field and the button. You can adjust the height value to increase or decrease the spacing as needed.
  • The submit-button class styles the button element occupy the full width of the form container, ensuring visual consistency with the input fields.
  • The button is styled with padding, background-color, text-color, and border-radius for a modern, clickable appearance.
  • The cursor:pointer; style changes the cursor to a pointer when hovering over the button, indicating it's clickable.
  • A hover effect is added (background-color: #45a049) to provide visual feedback when the user hovers over the button, enhancing the interactie experience.
