Controlling Element Visibility

We'll explore the essential properties of display and visibility, and how they allow you to manage the visibility of elements within your web page layout.

The display property provides options for altering how elements are displayed, while the visibility property offers control over an element's visibility without affecting its space within the layout.

Understanding the nuances of these properties is crucial for creating dynamic and responsive web desings.

Display property

The display property specifies if(or how) an element is displayed. Common values include none (element is completely removed), block, inline, and inline-block

The visibility property

The visibility property specifies whether an element is visible or not.. Unlike display:none, visibility:hidden keeps the element in the layout.

This div is always visible
This div is hidden but still takes up space.
This div is also visible

as a result, this div becomes invisible on the web page, but unlike display:none, it still occupies its original space in the layout.

This means the other elements on the page do not shift to fill the space that the hidden element would normally occupy, preservig the layout's overall structure.

display:none vs. visibility:hidden

When deciding between these two CSS properties:

  • Use display:none if you want an element to disappear entirely from the page layout, making it visible and removing it from the document flow.
  • Use visibility:hidden when you want to hide an element while keeping its space allocated in document layout.
