Hyperlink Properties

Understanding the five link states

  • Normal it hasn't been interacted with
  • Visited link enters the visited state.
  • Focus is related to the appearance of a link when it has focus.
  • Hover when a user hovers their mouse pointer over a link
  • Active when a user clicks on a link

Targetting Different Link States with Pseudo-Classes

Here are some commonly used pseudo-classes for styling links:

  • :link
  • :visited
  • :focus
  • :hover
  • :active

CSS Properties for Styling Links

  • color changes the text color of the link
  • text-decoration commonly used to add or remove underlines from links
  • background-color set the background color of the link.
  • font-weight changes the weight(boldness) of the link text.
  • border adds or modifies the border of the link
