The Line-Height Property

line-height sets the vertical spacing between lines of text. It's often used to improve readability

You can set it to various values, such as specific number (which is a multiple of the element's font size), normal (default line height), or inherit (inherits the line height from the parent element).

The Text-Shadow Property

text-shadow allows you to add a shadow effect to text, creating a visual separation between the text and its background. This property adds a shadow behind the text, providing depth and dimension to the typography. The shadow can be customized in terms of its color, position, and blur radius.

The syntax for the text-shadow property is as follows: 

text-shadow: horizontal-offset vertical-offset blur-radius color;
  • horizontal-offset specifies the horizontal distance the shadow will be offset from the text. Positive values move the shadow to the right, while negative values move it to the left
  • vertical-offset specifies the vertical distance the shadow will be offset from the text. Positive values move the shadow downwards, while negative values move it upwards.
  • blur-radius determines the amount of blurring applied to the shadow. A larger value creates a more diffused shadow, while a smaller value produces a sharper shadow.
  • color specifies the color of the shadow. This can be named color, hexadecimal code, and RGB value, or any other valid CSS color representation.
