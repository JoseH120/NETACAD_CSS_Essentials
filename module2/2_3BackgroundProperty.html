Background properties and gradient functions

Background properties

Building upon the foundation laid by the the background and background-color properties introduced in the previous section, let's now delve into yet another specific and specialized background property - background-image

The background-image property

The background-image property allows you to set an image as the background of an element. The image can be local or hosted online.

background-related properties

There is a number of background-related properties that allow you to modify the presentation of the background-image property. We'll discuss some of them

The background-repeat property

The background-repeat property determines how the background image should repeat (if it's smaller than the element)

Possible values include repeat (default - the background image is repeated both horizontally and vertically), repeat-x(the background image is repeated only horizontally), repeat-y (the background image is repeated only vertically), and no-repeat (the background image is not repeated)

The background-size property

the background-size property specifies the size of the background image

Possible values can be cover (the image is scaled to cover the entire container's background area, potentially cropping some parts of the image to maintain the aspect ratio) , contain (the image is scaled to fit entirely within the container's dimensions, preserving the aspect ratio. There might be empty space around the image), or specific dimensions like 200px 150px

The background-position property

The background-position property sets the starting position of the background image within its container.

You can use keywords (top, bottom, left, right) or precise length values (e.g 50%, 10px) to specify the horizontal and vertical position.

The background-attachment property

The background-attachment property controls whether a background image scrolls with the content or remains fixed as the user scrolls through the webpage.

The two possible values are scroll (the defualt) which scrolls the image with the content, and fixed which keeps the image stationary as the user scrolls.
