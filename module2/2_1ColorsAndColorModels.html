Colors and colors models

Colors and color models in CSS

there are several models that govern color manipulation, each offering a distinct approach to defining and utilizing colors. These are:

  1. RGB (Red Green Blue)
  2. HEX (Hexadecimal)
  3. HSL (Hue-Saturation-Ligthness)
  4. CMYK (Cyan-Magenta-Yellow-Key)
  5. HSV (Hue-Saturation-Value)

The RGB Model

As you might recall from the HTML Essentials course, colors are formed by combining differents levels of three primary colors: Red, Green and Blue

Additionally, you can use the RGBA syntax a variant of RGB - Which allows you to include an alpha channel for controlling transparency, which is essential for overlays and gradients and other design effects. The Alpha component (a) indicates the level of transparency of the color, raging from 0 (fully transparent) to 1 (fully opaque)

rgba(255, 0, 0, 1)

rgba(255, 0, 0, 0.75)

rgba(255, 0, 0, 0.50)

rgba(255, 0, 0, 0.25)

rgba(255, 0, 0, 0)

The HEX models

In the HEX model, colors are defined using a six-character-code, starting with a hash symbols

Red #FF0000

Green #00FF00

Blue #0000FF

The HSL model

The HSL model defines colors based on three main components: hue, saturation, and lightness

  1. Hue represents the actual color itsefls, raging from 0 to 360 degrees on the color wheel.Red is typically at 0 degrees, green at 120 degrees and blue at 240 degrees
  2. Saturation determines the intensity or purity of a color. A Saturation 0% results in a grayscale color (complete desaturated), while a saturation of 100% represents the most vibrant and intense version of the chosen hue.
  3. Lightness controls the brightness of the color. A lightness of 0% is completely black, while lightness of 100% is fully white

hsl(0,100%,50%)

hsl(120,100%,50%)

hsl(240,100%,50%)

Addionatlly, you can use the HSLA sintax to include an alpha value(opacity), allowing you to control the transparency of the color. For example

hsla(0,100%,50%,0.5)

hsla(60,30%,80%, 1)

The CMYK Model

The CMYK is a subtractive color model used primarly for color printing and reproduction. It stand for Cyan, Magenta, Yellow, and Key, representing the four ink plates used in the printing process

Unlike the additive RGB model, which involves mixing light to create colors, the CMYK model focuses on substracting light to achive the desired hues

While the CMYK model is primarly associated with printing, it's important to note that converting RGB colors to CMYK can lead to slight color shifts due to differences in color gamuts and ink limitations.

The HSV Model

The HSV color model is another way to define colors based on three main components: hue, saturation, and value.

The HSV model is particularly handy for creating color variations while maintaining the same brightness level. It's often used to generate shades and tints of a base color
