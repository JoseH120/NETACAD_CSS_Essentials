The filter property

filter allows for image manipulation, offering a range of effects like:

  • blur() - applies a blur effect to the element. You specify the blur radius as the function's argument. For example, filter: blur(5px); will blur the element with a 5-pixel radius
  • brightness() - adjust the brightness of the element. A value of 1 means no change, values less than 1 make it darker, and values greate than 1 make it brighter.
  • contrast() - alters the contrast of the element. Similar to brightness(), 1 means no chance, values less than 1 decrease contrast, and values greater than 1 increase contrast.
  • grayscale() - converts the element to grayscale. you can use grayscale(100%) to make the element completely grayscale.
  • invert() - inverts the colors of the element. A value of 1 (default) means no change, while 0 completely inverts the colors.
  • sepia() - applies a sepia tone effect to the element. a value of 100% makes the element completely sepia-toned

The z-index property

The z-index property determines their stacking order.

It's crucial for controlling which elements appear on top. It assings a numerical value to an element, indicating its position in the stacking order. Elements with higher z-index values will appear on top of elements with lower z-index values.
