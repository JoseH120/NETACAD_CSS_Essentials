Getting started with Flexbox

Controlling Layout With MainAxis And CrossAxis

  • Main Axix - The primary axis along which flex items are laid out. By default, it's horizontal
  • Cross Axis - perpendicular to the main axis. By default it's vertical

The flex container and flex items

Flexbox revolves around two key concepts: the flex containers and the flex items.

The flex container

you create a flex container by setting the display property of an element to flex (or inline-flex if you want inline layout). This elements becomes the flex container, and its children become the flex items

The flex-direction property

The flex-direction property determines the main axis of the layout, defining the direction flex items are placed in the flex container. It can be row, column, row-reverse, and column-reverse

The justify-content property

The justify-content property aligns flex items along the main axis(horizontal by default). It can be used to space out items or align them at the start, end, center, or distribute space between or around them.

The align-items property

The align-items property aligns flex items along the cross axis (perpendicular to the main axis).It's used for aligning items to the start, end, center, or stretching them to fill the container.

The flex-wrap property

By default, flex items try to fit into one line. The flex-wrap property allows the item to wrap as needed onto multiple lines, from top to bottom.

The flex items

Flex items are the children of the flex container and are the individual elements you want to lay out

Each flex item can also have properties applied to it, which define how it grows, shrinks, and how it should be aligned within the flex container.

The flex-grow property

The flex-grow property defines the ability for a flex item to growif necessary. It accepts a unitless value that serves as a proportion. It dictates what amount of the available space inside the flex container the item should take up.

The flex-shrink property

flex-shrink property defines the ability for a flex item to shrink if necessary. similarly to flex-grow, it accepts a unitless values that serves as a proportion.

The flex-basis property

flex-basis property defines the dafault the ability size of an element before the remaining space is distributed. It can be a length (like 20%, 5rem, etc) or a keyword like auto

The flex property

flex property is the shorthand for flex-grow, flex-shrink, and flex-basis combined. Its default value is 0 1 auto, but you can set it with one, two, or three values to control the item's flexibility

The align-self property

align-self property allows the default alingment (or the one specified by align-items) to be overridden for individual flex items.
