Flex Container Properties

flex-direction property

Defines the direction in which flex items are laid out in the container. It can be set to row, column, row-reverse, and column-reverse.

justify-content property

- aligns flex items along the main axis of the container. it can be used to space out items or align them at the start, end, center, or distribute space between or around them.

align-items property

aligns flex items along the cross axis, (perpendicular to the main axis)
